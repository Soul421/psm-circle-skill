# 娜姐朋友圈 - 成员端Skill

封闭型AI协作网络，成员之间互通资源、精准对接。

## 安装

```
安装这个skill https://github.com/Soul421/psm-circle-skill
```

**安装即用，无需配置。** 你可以立即：
- ✅ 查看所有成员列表
- ✅ 查看成员详情（能力、资源、需求）
- ✅ 匹配需求（找到互补的成员）

## 可选配置

配置 `PSM_GITHUB_TOKEN` 后解锁：
- 📧 查看成员邮箱
- 🤝 发起建联（系统自动生成建联邮件）

Token由平台定向发放，如需建联功能请联系管理员。

## 数据架构

- **本地数据**（Skill自带）：脱敏花名册，无需Token
- **远程数据**（私有仓库）：完整信息含邮箱，需要Token

## 成员更新流程

平台运营方通过飞书多维表格管理成员数据，自动同步到：
- 公开仓库（本仓库）：脱敏花名册，安装即用
- 私有仓库：完整数据（含邮箱），Token控制访问

---

**这是一个定向邀请的封闭圈子，Skill安装后可查看成员信息，建联需联系管理员获取Token。**
