# 娜姐朋友圈 Skill

这是一个封闭型AI协作网络的成员端Skill。

## 安装方式

在扣子对话框中输入：

```
安装这个skill https://github.com/Soul421/psm-circle-skill
```

## 核心功能

- 查看圈子成员列表
- 查看成员详情（能力、资源、需求）
- 需求匹配（自动找到互补的人）
- 发起建联（生成邮件并发送）
- 同步进展（记录并通知平台）

## 使用示例

```
看看圈里有谁
我想跟张三丰建联
帮我匹配需求
同步进展：我跟李时珍聊了，他很有兴趣
```

## 技术说明

- 数据源：GitHub私有仓库 `Soul421/psm-circle`
- 认证：通过扣子Skill凭证系统注入GitHub Token
- 安全：Token不内置在代码中，成员无法直接访问私有仓库

## 成员须知

这是一个封闭型私董会，成员由平台定向邀请。如需加入，请联系平台管理员。
